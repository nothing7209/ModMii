Check Mii Out Channel

The Check Mii Out Channel allowed users to share their customized Mii avatars with a global community. Although Nintendo officially discontinued the service in 2013, fan-made projects like WiiLink have brought its functionality back.

Everybody Votes Channel

The Everybody Votes Channel allowed users to participate in global polls. Cast your vote and predict results! Nintendo discontinued the service in 2013, but it remains a classic Wii experience.

Mii Channel

The Mii Channel is a official Channel where you can create your own figures called "Miis". This Software is pre-installed on every (official) Nintendo Wii (or vWii) Console.

Homebrew Channel

The Homebrew Channel is a free application that lets users run unofficial software and games on their Nintendo Wii. It unlocks hidden potential and custom content not supported by Nintendo.

Internet Channel

The Internet Channel was a browser for the Wii powered by Opera. It allowed users to browse the web on their TV using the Wii Remote. Support was discontinued in 2013.

WiiMart

WiiMart is an unofficial revival for the Wii Shop Channel. WARNING: To install this Software on any Console or Emulator it required the System to be Homebrewed.
